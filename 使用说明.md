# moyu 使用说明 🐟

在终端里摸鱼的知乎客户端:看推荐 / 热榜 / 搜索 / 回答 / 评论,全程方向键。自带**老板键**和 **Claude Code 风格伪装**,低调摸鱼。

---

## 一、安装

### 方式 A:npm(推荐,一行搞定)

> 需要先装 [Node.js](https://nodejs.org)(16+)。安装时会自动下载对应你系统(Windows / macOS / Linux)的可执行文件。

```bash
npm install -g moyu-fish
```

装完直接运行:

```bash
moyu
```

更新 / 卸载:

```bash
npm update -g moyu-fish     # 更新
npm uninstall -g moyu-fish  # 卸载
```

### 方式 B:直接下载可执行文件

去 [Releases 页面](https://github.com/ItChinaW/Moyu-Cmd-Claude-Style/releases) 下载对应你系统的文件:

| 系统 | 下载 |
|------|------|
| **Windows** | `moyu-win32-x64.exe`(可重命名为 `moyu.exe`) |
| **macOS**(Intel / Apple Silicon 通用) | `moyu-darwin-universal` |

它是**单文件**,不依赖任何东西,放哪都能跑。

#### macOS 首次运行被拦(「无法打开,来自身份不明的开发者」)

二进制没做苹果签名,系统会拦一下。任选其一解除:

```bash
# 终端里执行(把路径换成你下载的文件)
xattr -d com.apple.quarantine ~/Downloads/moyu-darwin-universal
chmod +x ~/Downloads/moyu-darwin-universal
~/Downloads/moyu-darwin-universal
```

或者:在访达里**右键点文件 → 打开 → 再点「打开」**,之后就不再拦了。

#### Windows 首次运行被拦(SmartScreen 蓝色弹窗)

点 **「更多信息」 → 「仍要运行」** 即可(因为没做代码签名,首次会提示)。

## 二、第一次使用:粘贴知乎 Cookie

启动后,在底部 `>` 命令行输入 `/zhihu` 回车。第一次会让你粘贴知乎 Cookie:

1. 用浏览器登录 [zhihu.com](https://www.zhihu.com)
2. 按 **F12** 打开开发者工具 → 切到 **Network(网络)** 标签
3. 刷新页面,点任意一个 `www.zhihu.com` 的请求
4. 在 **Request Headers(请求标头)** 里找到 `cookie:` 那一行,**复制整行的值**(很长)
5. 回到 moyu,粘贴到 `>` 后面回车

验证通过后会自动保存到本地,以后启动直接进推荐流,不用再粘。

> Cookie 等于你的知乎登录态,**明文保存在本地**,别发给别人:
> - macOS:`~/Library/Application Support/touch-fish/config.toml`
> - Windows:`%APPDATA%\touch-fish\config.toml`
> - Linux:`~/.config/touch-fish/config.toml`

## 三、操作

底部平时只有一个 `>` 提示符(低调)。输入 `/help` 回车可随时查看下面这些。

### 命令(在 `>` 后输入)

| 命令 | 作用 |
|------|------|
| `/zhihu` | 进入知乎(推荐流 · 最新) |
| `/hot` | 热榜 |
| `/search 关键词` | 搜索 |
| `/refresh` | 刷新当前列表(也可按 `r`) |
| `/login` | 重新登录 / 切换账号(粘贴新 Cookie) |
| `/help` | 显示帮助 |
| `/quit` | 退出 |

### 按键

| 按键 | 作用 |
|------|------|
| `↑` `↓` | 选择 / 滚动 |
| `Enter` | 进入选中项 / 执行命令 |
| `→` 或 `Tab` | 看评论(详情页) |
| `←` 或 `Esc` | 返回上一级 |
| `n` / `p` | 上一个 / 下一个回答(详情页) |
| `1`–`9` | 在浏览器打开第 N 张图(详情页) |
| `c` | 开关 Claude 伪装(详情页) |
| `r` | 刷新(列表页) |
| `q` | 退出 |
| **`` ` `` 或 `·`** | **老板键:一键隐藏 / 恢复**(中英文输入法都行) |

## 四、摸鱼特性

- **老板键**:按左上角那个键(英文 `` ` ``、中文输入法下是 `·`),界面瞬间变成一屏「正在跑构建」的假日志,再按一下切回来。
- **Claude 伪装**:看回答时,正文段落之间穿插逼真的 Claude Code 工具调用(`● Bash(...)`、`Error: Exit code`、git diff……),真实内容是亮色照常读;按 `c` 可关。
- **无边框低调界面**:远看就是一片普通终端文字。
