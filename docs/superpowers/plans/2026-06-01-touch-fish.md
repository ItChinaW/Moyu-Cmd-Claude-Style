# touch-fish Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a terminal "摸鱼" tool that browses Zhihu (hot list, search, answer detail, comments) inside the cmd window using slash-commands plus arrow-key navigation.

**Architecture:** A single Rust binary. A platform-agnostic data layer (`config`, `net`, `platform::zhihu`) produces typed models; the `x-zse-96` request signature is generated by running the vendored obfuscated `zhihu.raw.js` inside an embedded QuickJS engine (`rquickjs`). A `ratatui` + `crossterm` async event loop (tokio) renders screens and routes keys/commands through a central `App` state machine.

**Tech Stack:** Rust, ratatui, crossterm, tokio, reqwest (rustls), serde/serde_json, rquickjs (QuickJS), toml, dirs, unicode-width, scraper, anyhow/thiserror.

**Reference:** Zhihu endpoints + signature scheme are ported from `ylw1997/touchFish`. Source spec: `docs/superpowers/specs/2026-06-01-touch-fish-design.md`.

---

## Phases (each phase ends with runnable software)

- **Phase 0** — Scaffolding + signature spike (resolves the highest-risk unknown first).
- **Phase 1** — Data layer: config, signer, net, Zhihu client, HTML→text. Ends with a `--debug-hot` CLI that prints the real hot list.
- **Phase 2** — TUI: state machine, command parser, rendering, async loop, login flow. Ends with the full interactive app.

> **Network-dependent tests** are integration tests gated behind the `ZHIHU_COOKIE` env var. They `eprintln!` a skip notice and `return` early when it is unset, so CI without a cookie stays green. Pure logic is unit-tested normally.

```rust
use anyhow::{anyhow, Context, Result};
use rquickjs::{Context as JsContext, Function, Runtime};

const CRYPTO_JS: &str = include_str!("../../../assets/crypto-js.min.js");
const ZHIHU_JS: &str = include_str!("../../../assets/zhihu.raw.js");

/// Wraps a QuickJS context that has the Zhihu `encrypt` function loaded.
pub struct ZhihuSigner {
    ctx: JsContext,
}

impl ZhihuSigner {
    pub fn new() -> Result<Self> {
        let rt = Runtime::new().context("create quickjs runtime")?;
        let ctx = JsContext::full(&rt).context("create quickjs context")?;
        ctx.with(|ctx| -> Result<()> {
            let g = ctx.globals();
            // Browser-ish globals the signer pokes at.
            ctx.eval::<(), _>("var window = window || {}; var self = window; var navigator = { userAgent: 'Mozilla/5.0' };")
                .map_err(|e| anyhow!("init globals: {e}"))?;
            // Provide require('crypto-js'): eval the UMD into a module, hand it back.
            ctx.eval::<(), _>(format!(
                "var __cryptojs_module = {{ exports: {{}} }};\n(function(module, exports){{\n{CRYPTO_JS}\n}})(__cryptojs_module, __cryptojs_module.exports);"
            ).as_str()).map_err(|e| anyhow!("eval crypto-js: {e}"))?;
            g.set("require", Function::new(ctx.clone(), |name: String| -> rquickjs::Result<rquickjs::Value> {
                // resolved below via closure-free path; see Step 3 note
                let _ = name; unreachable!()
            }).ok()).ok();
            Ok(())
        })?;
        Ok(Self { ctx })
    }
}
```

> NOTE: closures capturing JS values across `with` boundaries are awkward in rquickjs. Step 3 replaces the `require` stub with a pure-JS shim (simpler and avoids Rust closure lifetime issues). Keep Step 1's file compiling by removing the `g.set("require", ...)` line — it's shown here only to motivate Step 3.

```rust
impl ZhihuSigner {
    pub fn new() -> Result<Self> {
        let rt = Runtime::new().context("create quickjs runtime")?;
        let ctx = JsContext::full(&rt).context("create quickjs context")?;
        ctx.with(|ctx| -> Result<()> {
            // Browser-ish globals.
            ctx.eval::<(), _>(
                "var window = (typeof window!=='undefined')?window:{}; var self = window; \
                 var navigator = { userAgent: 'Mozilla/5.0' }; var document = {};",
            ).map_err(|e| anyhow!("init globals: {e}"))?;
            // crypto-js as a CommonJS module, then a require() that returns it by name.
            ctx.eval::<(), _>(format!(
                "var __mods = {{}};\n\
                 (function(){{ var module={{exports:{{}}}}; var exports=module.exports;\n{CRYPTO_JS}\n __mods['crypto-js']=module.exports; }})();\n\
                 function require(n){{ if(__mods[n]) return __mods[n]; throw new Error('module not found: '+n); }}"
            ).as_str()).map_err(|e| anyhow!("eval crypto-js + require: {e}"))?;
            // The signer itself, captured into a CommonJS module.
            ctx.eval::<(), _>(format!(
                "var __zhihu = {{ exports: {{}} }};\n\
                 (function(module, exports){{\n{ZHIHU_JS}\n}})(__zhihu, __zhihu.exports);\n\
                 var __encrypt = __zhihu.exports.encrypt;\n\
                 if (typeof __encrypt !== 'function') throw new Error('no encrypt export');"
            ).as_str()).map_err(|e| anyhow!("eval zhihu.raw.js: {e}"))?;
            Ok(())
        })?;
        Ok(Self { ctx })
    }

    /// Run the JS `encrypt(input)` and return the `x-zse-96` value.
    pub fn sign(&self, input: &str) -> Result<String> {
        self.ctx.with(|ctx| -> Result<String> {
            ctx.globals().set("__sign_input", input)
                .map_err(|e| anyhow!("set input: {e}"))?;
            let out: String = ctx.eval("__encrypt(__sign_input)")
                .map_err(|e| anyhow!("call encrypt: {e}"))?;
            Ok(out)
        })
    }
}
```

```rust
impl Config {
    pub fn config_path() -> PathBuf {
        dirs::config_dir()
            .unwrap_or_else(|| PathBuf::from("."))
            .join("touch-fish")
            .join("config.toml")
    }

    pub fn load() -> Result<Self> {
        Self::load_from(&Self::config_path())
    }

    pub fn load_from(path: &std::path::Path) -> Result<Self> {
        match std::fs::read_to_string(path) {
            Ok(s) => toml::from_str(&s).context("parse config.toml"),
            Err(e) if e.kind() == std::io::ErrorKind::NotFound => Ok(Self::default()),
            Err(e) => Err(e).context("read config.toml"),
        }
    }

    pub fn save(&self) -> Result<()> {
        self.save_to(&Self::config_path())
    }

    pub fn save_to(&self, path: &std::path::Path) -> Result<()> {
        if let Some(parent) = path.parent() {
            std::fs::create_dir_all(parent).context("create config dir")?;
        }
        let s = toml::to_string_pretty(self).context("serialize config")?;
        std::fs::write(path, s).context("write config.toml")
    }
}
```

```rust
/// Extract a cookie field's value from a raw `Cookie:` header string.
/// Surrounding double-quotes (Zhihu wraps `d_c0` in quotes) are stripped.
pub fn field(cookie: &str, name: &str) -> Option<String> {
    for pair in cookie.split(';') {
        let pair = pair.trim();
        if let Some((k, v)) = pair.split_once('=') {
            if k.trim() == name {
                return Some(v.trim().trim_matches('"').to_string());
            }
        }
    }
    None
}
```

```rust
#[derive(Clone)]
pub struct HttpClient {
    inner: reqwest::Client,
}

impl HttpClient {
    pub fn new() -> Result<Self> {
        let inner = reqwest::Client::builder()
            .gzip(true)
            .build()
            .context("build reqwest client")?;
        Ok(Self { inner })
    }

    /// GET `{HOST}{path_with_query}` with signed Zhihu headers. Returns body text.
    pub async fn signed_get(
        &self,
        path_with_query: &str,
        cookie: &str,
        x_zse_96: &str,
    ) -> Result<String> {
        let url = format!("{HOST}{path_with_query}");
        let headers = zhihu_headers(cookie, x_zse_96)?;
        let resp = self.inner.get(&url).headers(headers).send().await
            .context("send request")?;
        let status = resp.status();
        let body = resp.text().await.context("read body")?;
        if !status.is_success() {
            anyhow::bail!("HTTP {status}: {}", body.chars().take(200).collect::<String>());
        }
        Ok(body)
    }
}
```

```rust
use scraper::Html;

/// Convert Zhihu answer/comment HTML into plain terminal text.
/// `<p>` → blank-line-separated blocks, `<br>` → newline, `<img>` → `[图片]`.
pub fn to_text(html: &str) -> String {
    let frag = Html::parse_fragment(html);
    let mut out = String::new();
    for edge in frag.tree.root().traverse() {
        if let Edge::Open(node) = edge {
            match node.value() {
                Node::Text(t) => out.push_str(t),
                Node::Element(e) => match e.name() {
                    "br" => out.push('\n'),
                    "img" => out.push_str("[图片]"),
                    "p" if !out.is_empty() => out.push_str("\n\n"),
                    _ => {}
                },
                _ => {}
            }
        }
    }
    out.trim().to_string()
}
```

```rust
use anyhow::{anyhow, Result};
use crate::net::HttpClient;
use crate::platform::{ListEntry, DetailView, CommentView};
use crate::platform::zhihu::{cookie, html, model, sign::{self, ZhihuSigner}};

pub struct ZhihuClient {
    cookie: String,
    d_c0: String,
    signer: ZhihuSigner,
    http: HttpClient,
}

impl ZhihuClient {
    pub fn new(cookie: String) -> Result<Self> {
        let d_c0 = cookie::field(&cookie, "d_c0")
            .ok_or_else(|| anyhow!("cookie missing d_c0 — re-copy from browser"))?;
        Ok(Self { cookie, d_c0, signer: ZhihuSigner::new()?, http: HttpClient::new()? })
    }

    async fn get(&self, path: &str) -> Result<String> {
        let sig = self.signer.sign(&sign::build_sign_input(path, &self.d_c0))?;
        self.http.signed_get(path, &self.cookie, &sig).await
    }

    pub async fn hot_list(&self) -> Result<Vec<ListEntry>> {
        let body = self.get("/api/v3/feed/topstory/hot-lists/total?limit=50&desktop=true").await?;
        let resp: model::HotListResponse = serde_json::from_str(&body)?;
        Ok(resp.data.into_iter().map(|it| {
            let qid = it.target.url.rsplit('/').find_map(|s| s.parse::<u64>().ok());
            ListEntry { title: it.target.title, subtitle: it.detail_text, question_id: qid }
        }).collect())
    }

    pub async fn search(&self, query: &str) -> Result<Vec<ListEntry>> {
        let enc = urlencode(query);
        let path = format!("/api/v4/search_v3?t=general&q={enc}&offset=0&limit=20");
        let body = self.get(&path).await?;
        let resp: model::SearchResponse = serde_json::from_str(&body)?;
        Ok(resp.data.into_iter().filter_map(|it| {
            let title = it.highlight.map(|h| h.title).filter(|t| !t.is_empty())
                .unwrap_or(it.object.title);
            let qid = it.object.question.map(|q| q.id).filter(|&id| id != 0);
            if title.is_empty() { return None; }
            Some(ListEntry { title: strip_em(&title), subtitle: it.object.kind, question_id: qid })
        }).collect())
    }

    pub async fn answers(&self, question_id: u64) -> Result<Vec<DetailView>> {
        let path = format!(
            "/api/v4/questions/{question_id}/feeds?include=data%5B%2A%5D.content&limit=10&offset=0"
        );
        let body = self.get(&path).await?;
        let resp: model::AnswersResponse = serde_json::from_str(&body)?;
        Ok(resp.data.into_iter().map(|f| DetailView {
            title: String::new(),
            author: f.target.author.name,
            voteup: f.target.voteup_count,
            body: html::to_text(&f.target.content),
            answer_id: f.target.id,
        }).collect())
    }

    pub async fn comments(&self, answer_id: u64) -> Result<Vec<CommentView>> {
        let path = format!(
            "/api/v4/comment_v5/answers/{answer_id}/root_comment?order_by=score&limit=100"
        );
        let body = self.get(&path).await?;
        let resp: model::CommentResponse = serde_json::from_str(&body)?;
        Ok(resp.data.into_iter().map(|c| CommentView {
            author: c.author.member.name,
            body: html::to_text(&c.content),
            like_count: c.like_count,
            child_count: c.child_comment_count,
        }).collect())
    }
}

fn urlencode(s: &str) -> String {
    s.bytes().map(|b| match b {
        b'a'..=b'z' | b'A'..=b'Z' | b'0'..=b'9' | b'-' | b'_' | b'.' | b'~' => (b as char).to_string(),
        _ => format!("%{b:02X}"),
    }).collect()
}

fn strip_em(s: &str) -> String { s.replace("<em>", "").replace("</em>", "") }
```

```rust
/// Parse a slash-command line into a `Command`.
pub fn parse(line: &str) -> Command {
    let line = line.trim();
    let (head, rest) = match line.split_once(char::is_whitespace) {
        Some((h, r)) => (h, r.trim()),
        None => (line, ""),
    };
    match head {
        "/zhihu" => Command::Zhihu,
        "/login" => Command::Login,
        "/back" => Command::Back,
        "/quit" => Command::Quit,
        "/search" if !rest.is_empty() => Command::Search(rest.to_string()),
        _ => Command::Unknown(line.to_string()),
    }
}
```

```rust
pub enum Request {
    Connect(String),          // cookie
    HotList,
    Search(String),
    Answers(u64),
    Comments(u64),
}

pub enum Update {
    Connected,
    ConnectFailed(String),
    List(Vec<ListEntry>),
    Details(Vec<DetailView>),
    Comments(Vec<CommentView>),
    Error(String),
}

/// Worker thread: owns the !Send ZhihuClient, serves requests.
fn spawn_worker(mut rx: mpsc::UnboundedReceiver<Request>, tx: mpsc::UnboundedSender<Update>) {
    std::thread::spawn(move || {
        let rt = tokio::runtime::Builder::new_current_thread().enable_all().build().unwrap();
        rt.block_on(async move {
            let mut client: Option<ZhihuClient> = None;
            while let Some(req) = rx.recv().await {
                let upd = handle(&mut client, req).await;
                if tx.send(upd).is_err() { break; }
            }
        });
    });
}

async fn handle(client: &mut Option<ZhihuClient>, req: Request) -> Update {
    match req {
        Request::Connect(cookie) => match ZhihuClient::new(cookie) {
            Ok(c) => match c.hot_list().await {
                Ok(list) => { *client = Some(c); Update::List(list) }
                Err(e) => Update::ConnectFailed(e.to_string()),
            },
            Err(e) => Update::ConnectFailed(e.to_string()),
        },
        Request::HotList => run(client, |c| c.hot_list(), Update::List).await,
        Request::Search(q) => run(client, move |c| c.search(&q), Update::List).await,
        Request::Answers(id) => run(client, move |c| c.answers(id), Update::Details).await,
        Request::Comments(id) => run(client, move |c| c.comments(id), Update::Comments).await,
    }
}

async fn run<T, F, Fut>(client: &mut Option<ZhihuClient>, f: F, ok: fn(T) -> Update) -> Update
where F: FnOnce(&ZhihuClient) -> Fut, Fut: std::future::Future<Output = anyhow::Result<T>> {
    match client {
        Some(c) => match f(c).await { Ok(v) => ok(v), Err(e) => Update::Error(e.to_string()) },
        None => Update::Error("未登录".into()),
    }
}
```

```rust
pub async fn run_app(cookie: String) -> Result<()> {
    crossterm::terminal::enable_raw_mode()?;
    let mut stdout = std::io::stdout();
    crossterm::execute!(stdout, crossterm::terminal::EnterAlternateScreen)?;
    let mut term = Terminal::new(CrosstermBackend::new(stdout))?;

    let (req_tx, req_rx) = mpsc::unbounded_channel::<Request>();
    let (upd_tx, mut upd_rx) = mpsc::unbounded_channel::<Update>();
    spawn_worker(req_rx, upd_tx);

    let mut app = App::new();
    app.cookie = cookie.clone();
    if cookie.is_empty() {
        app.replace(Screen::Login);
    } else {
        app.loading = true;
        let _ = req_tx.send(Request::Connect(cookie));
    }

    let mut events = EventStream::new();
    let mut tick = tokio::time::interval(Duration::from_millis(120));

    loop {
        term.draw(|f| crate::ui::draw(f, &app))?;
        if app.should_quit { break; }
        tokio::select! {
            maybe_ev = events.next() => {
                if let Some(Ok(Event::Key(k))) = maybe_ev {
                    if k.kind == KeyEventKind::Press {
                        handle_key(&mut app, k.code, &req_tx);
                    }
                }
            }
            Some(upd) = upd_rx.recv() => apply_update(&mut app, upd),
            _ = tick.tick() => {}
        }
    }

    crossterm::terminal::disable_raw_mode()?;
    crossterm::execute!(term.backend_mut(), crossterm::terminal::LeaveAlternateScreen)?;
    Ok(())
}

fn apply_update(app: &mut App, upd: Update) {
    app.loading = false;
    match upd {
        Update::Connected | Update::List(_) if matches!(upd, Update::Connected) => {}
        Update::List(list) => { app.error = None; app.set_list(list); if *app.screen() != Screen::List { app.push(Screen::List); } }

        Update::Connected => {}
        Update::ConnectFailed(e) => { app.error = Some(e); app.replace(Screen::Login); }
        Update::Error(e) => { app.error = Some(e); }
    }
}
```

```rust
fn handle_key(app: &mut App, code: KeyCode, req_tx: &mpsc::UnboundedSender<Request>) {
    use crate::app::state::Screen::*;
    // Command line has priority when the user is typing one.
    match code {
        KeyCode::Char(c) => {
            // On Login screen everything typed is the cookie; elsewhere only after '/'.
            if *app.screen() == Login || c == '/' || !app.command.is_empty() {
                app.command.push(c);
                return;
            }
            // shortcuts when not composing a command
            if c == 'q' { app.should_quit = true; }
            return;
        }
        KeyCode::Backspace => { app.command.pop(); return; }
        KeyCode::Esc => { app.command.clear(); app.back(); return; }
        KeyCode::Enter => {
            if *app.screen() == Login {
                let cookie = std::mem::take(&mut app.command);
                if !cookie.is_empty() {
                    app.cookie = cookie.clone();
                    app.loading = true;
                    let _ = req_tx.send(Request::Connect(cookie));
                }
                return;
            }
            if !app.command.is_empty() {
                let cmd = command::parse(&std::mem::take(&mut app.command));
                dispatch_command(app, cmd, req_tx);
                return;
            }
            // Enter with empty command = open selection
            open_selection(app, req_tx);
            return;
        }
        KeyCode::Up => match app.screen() {
            List => app.cursor_up(),
            Detail => app.detail_scroll = app.detail_scroll.saturating_sub(1),
            Comments => app.comment_scroll = app.comment_scroll.saturating_sub(1),
            _ => {}
        },
        KeyCode::Down => match app.screen() {
            List => app.cursor_down(),
            Detail => app.detail_scroll = app.detail_scroll.saturating_add(1),
            Comments => app.comment_scroll = app.comment_scroll.saturating_add(1),
            _ => {}
        },
        KeyCode::Left => app.back(),
        KeyCode::Right | KeyCode::Tab => {
            if *app.screen() == Detail {
                if let Some(d) = app.current_detail() {
                    app.loading = true;
                    let _ = req_tx.send(Request::Comments(d.answer_id));
                }
            }
        }
        _ => {}
    }
}

fn open_selection(app: &mut App, req_tx: &mpsc::UnboundedSender<Request>) {
    if *app.screen() == crate::app::state::Screen::List {
        if let Some(e) = app.selected_entry() {
            if let Some(qid) = e.question_id {
                app.loading = true;
                let _ = req_tx.send(Request::Answers(qid));
            }
        }
    }
}

fn dispatch_command(app: &mut App, cmd: Command, req_tx: &mpsc::UnboundedSender<Request>) {
    use crate::app::state::Screen;
    match cmd {
        Command::Zhihu => {
            if app.cookie.is_empty() { app.replace(Screen::Login); }
            else { app.loading = true; let _ = req_tx.send(Request::HotList); }

        Command::Search(q) => { app.loading = true; let _ = req_tx.send(Request::Search(q)); }

        Command::Back => app.back(),
        Command::Quit => app.should_quit = true,
        Command::Unknown(s) => app.error = Some(format!("未知命令: {s}")),
    }
}
```
